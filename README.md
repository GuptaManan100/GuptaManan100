## Hi there 👋

I’m **Manan Gupta**, a software engineer specializing in scalable distributed database systems and open-source contributions. I completed my bachelor’s in computer science from the Indian Institute of Technology Guwahati and have been actively involved in open-source database projects.

---

### Key Projects

- **VTOrc (High-Availability for Vitess):**  
  Led the creation of VTOrc by forking Orchestrator, implementing automatic failover logic, monitoring capabilities, and seamless recovery mechanisms to ensure minimal downtime for Vitess clusters.

- **Vitess Query Serving Enhancements:**  
  Added robust foreign key constraint support within Vitess, enabling relational integrity checks, and implemented atomic distributed transactions to guarantee consistency across sharded environments.

- **Automated Cherry-Picks Between Repositories:**  
  Developed tooling and workflows to automatically generate and apply cherry‐picks between the open‐source Vitess repository and our private enterprise fork, significantly reducing manual merge overhead and sync latency.

- **Vitess Operator Maintenance:**  
  Continued development and upkeep of the Vitess Kubernetes Operator.

- **LFX Mentorship:**  
  Acted as an LFX (Linux Foundation) mentor, providing guidance to multiple open-source contributors, helping them contribute to Vitess on various projects like the parser.
